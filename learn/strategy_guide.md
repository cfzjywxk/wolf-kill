# 狼人杀 6 人局赛前策略知识库

适用局：`2 狼人 + 预言家 + 女巫 + 2 平民`。
目标：让 AI 更像会玩的竞技玩家，而不是空发言、乱定身份、乱散票。

## 1. 最高优先级规则

- 只能根据 **当前可见信息** 推理；绝不能把隐藏夜间信息带到公开发言里。
- 白天发言必须给出：**主怀疑位 / 理由 / 今天主推 / 对后置位的问题**。
- 不要只说“先听听”“信息少”“我先观望”；这类空话会严重降低好人胜率。
- 投票必须服务推进；**散票、弃票、只点评不落票**，通常更帮狼人。

## 2. 六人局结构意识

- 6 人局容错极低，**第 1 天发言质量就能决定胜负**。
- 不能把“死亡结果”直接当翻牌信息。本局公开只会告诉你谁死了，不会公开死因与身份。
- **双死 ≠ 必然一狼一神 / 一狼一民**；只能结合是否终局、查验、发言、票型继续缩坑。
- **游戏没结束** 只说明“狼和好人都还没达成胜利条件”，**不能直接反推死者的精确身份组合**。
- 不要把“更像被刀”“更像被毒”说成已经确定；必须明确这是倾向判断。

## 3. 好人通用打法

- 核心任务：找狼、逼观点、集中票型。
- 优先盘 **公开事实**：发言逻辑、投票是否一致、死亡顺序、查验结果、是否回避关键问题。
- 常用方法：
  - **PoE（排除法）**：先找相对做好的人，再缩小狼坑。
  - **Associatives（关联分析）**：看谁在保谁、踩谁、跟票谁，但不要过度脑补。
  - **NKA（夜刀分析）**：夜刀只提供辅助信息，不能单独定身份。
- 好人发言要逼后位回答具体问题，例如：
  - “你今天唯一主推谁？”
  - “你认为谁的视角最像狼，为什么？”
  - “如果你站这个边，明天怎么延续票型？”

## 4. 预言家

- 查验优先：**高争议位 > 影响票型位 > 对跳位**。
- 跳身份时不要只报结果，要同时说清：
  - 为什么验这个人；
  - 今天应该推谁；
  - 如果你出局，好人明天按什么逻辑继续。
- 面对对跳，重点比：
  - 验人理由是否自然；
  - 前后视角是否一致；
  - 有没有真实推进路线，而不是只抢话语权。

## 5. 女巫

- 本项目规则：**女巫不能自救**。
- 解药不是见刀必救；要看收益：救了以后白天能不能形成推进。
- 毒药要打 **高置信狼**，优先毒：视角漏洞大、发言动机差、票型与发言冲突的人。
- 女巫白天通常不要轻易裸跳，除非能直接定票或阻止狼人控场。

## 6. 平民

- 平民不是“没信息的人”，而是要负责补齐白天逻辑。
- 首轮发言尽量包含：
  - 你当前最怀疑谁；
  - 一个暂时不想推的人；
  - 对前置位至少一条直接回应；
  - 给后置位的具体问题。
- 即使信息少，也要指出：谁在空发言，谁在偷换概念，谁投票和发言不一致。

## 7. 狼人团队打法

### 夜间密谈必须快速收束

至少明确四件事：
- 今晚主刀谁、备刀谁；
- 明天谁适合悍跳；
- 出现对跳时谁冲锋、谁倒钩、谁深水；
- 若无新内容，直接说“无更多讨论”，不要灌水。

### 常见战术

- **Counterclaim / 对跳**：假跳预言家时，必须有完整验人理由与白天推进，不要只报结论。
- **Bussing / 狼踩狼**：可用于做身份，但不能踩到真把队友送走却没有后手。
- **Distancing / 拉开距离**：与队友保持自然分歧，避免被轻易做成同边。
- **倒钩**：顺着真预言家或好人逻辑卖队友，换后续生存空间。
- **深水**：少说但不空；“低调”不等于没立场。

### 狼人常见失误

- 在白天暴露夜间视角；
- 悍跳没有验人逻辑，只有结论；
- 狼队白天口径不一致；
- 一边说“没信息”，一边又强行锁定全局。

## 8. WIFOM 使用边界

- **WIFOM**（“他会不会故意这么演”）只能少量参考，不能无限空转。
- 当你开始过度读心时，强制回到三个问题：
  - 这段发言帮助了哪个阵营？
  - 这张票是否符合他之前的立场？
  - 这个人有没有给出可执行的推进方案？

## 9. 推荐输出模板

### 白天发言

按这个顺序组织：**结论 → 依据 → 回应 → 推进**。

### 狼人密谈

按这个顺序组织：**刀口建议 → 明天分工 → 对跳预案 → 是否还有补充**。
